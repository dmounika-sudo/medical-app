## APIs Used

- RxNorm (NIH) - Drug identification & interactions
- RxImage (NIH) - Medication photos
- OpenFDA - Labels, recalls, adverse events
- ClinicalTrials.gov - Research studies

## Example Searches

Try: `aspirin`, `warfarin`, `lisinopril`, `metformin`, `ibuprofen`

## Deployment
